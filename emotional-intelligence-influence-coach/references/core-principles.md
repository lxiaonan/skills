# Core Principles

## 1. 情绪是信息，不是命令

情绪提示需求、边界、价值和风险。教练应帮助用户识别情绪，而不是否定情绪或让情绪直接指挥行为。

Use prompts:

- “你现在最强烈的情绪是什么？愤怒、委屈、害怕、羞耻、焦虑还是失望？”
- “这个情绪在提醒你什么需求没有被满足？”
- “如果先不行动，10分钟后你希望自己怎样处理？”

## 2. 先自我觉察，再影响他人

高压沟通中的第一目标不是立刻说服别人，而是避免被情绪带着走。稳定自己后，用户才能听清对方、表达清楚、守住边界。

Core distinction:

- 可控：自己的语气、时机、问题、请求、边界、记录、是否继续对话。
- 不可控：对方立刻认同、道歉、喜欢自己、改变性格、承认全部错误。

## 3. 先复述，再立场

当对方情绪强烈时，直接解释容易被听成辩解。先复述对方关切，再表达自己的立场，能降低防御。

Pattern:

1. “我听到你主要担心的是……”
2. “如果我理解错了，请你纠正。”
3. “我的情况/看法是……”
4. “我希望我们下一步……”

## 4. 用事实-感受-需求-请求替代指责

Blame pattern:

- “你总是不尊重我。”
- “你就是故意针对我。”

Constructive pattern:

- 事实：“在今天会议上，我汇报时被打断了三次。”
- 感受：“我有些挫败，也担心信息没说完整。”
- 需求：“我需要有完整表达方案的机会。”
- 请求：“下次我先用5分钟讲完，再集中回应问题，可以吗？”

## 5. 同理不等于同意

同理是理解对方感受和处境，不是放弃自己的判断或边界。

Useful wording:

- “我能理解这件事让你很着急。”
- “我理解你的顾虑，但我不能接受人身攻击。”
- “我愿意继续谈问题本身，不继续互相指责。”

## 6. 影响力来自信任账户

影响力不是单次话术，而是长期信任积累。信任来自：兑现承诺、尊重对方、承认贡献、透明沟通、在冲突中不羞辱对方。

Build trust through:

- 具体认可，而不是空泛夸奖。
- 先理解对方目标，再提出自己的方案。
- 承认对方合理顾虑。
- 给对方选择空间。
- 事后跟进并兑现。

## 7. 宽容不是纵容

宽容是减少报复和羞辱，给关系修复空间；纵容是放弃底线、允许伤害重复发生。

Boundary language:

- “我愿意听你的不满，但不能接受辱骂。”
- “如果继续用这个语气，我们先暂停，下午再谈。”
- “我可以帮忙一次，但这不能成为长期默认安排。”

## 8. 赞美要具体、真实、适度

认可和赞美能降低防御、增强合作，但虚假恭维会破坏信任。

Effective praise model:

1. 说出具体行为：“你刚才把客户的担心整理成三点。”
2. 说明价值：“这让团队更快定位问题。”
3. 鼓励重复：“以后重大投诉也可以先这样结构化。”

## 9. 自信是可训练的行为状态

自信不是保证自己一定赢，而是相信自己可以面对、学习、调整。用户可通过准备、正向行动、身体状态、复盘和小胜利来建立自信。

Useful coaching:

- “先把你能控制的三件事列出来。”
- “准备一句开场、一句澄清、一句边界。”
- “目标不是完美，而是比上一次更稳。”

## 10. 沟通目标优先于情绪胜负

在冲突中，很多人想证明自己对。教练应帮助用户回到目标：保护关系、解决问题、守住边界、推进下一步、留存证据或安全退出。
