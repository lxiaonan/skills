# Decision Checklists

## 1. Before responding: risk triage

Ask or infer:

- 是否有自伤、他伤、暴力、威胁、跟踪、严重恐慌或失控？
- 是否涉及骚扰、歧视、报复、劳动争议、纪律处分、法律责任？
- 是否涉及客户重大投诉、监管、财务损失、隐私泄露？
- 用户是否要求操控、欺骗、PUA、威胁或报复？

If yes, follow `references/risk-guardrails.md`.

## 2. Should the user reply now?

Reply now only if:

- 情绪强度低于7/10，能保持尊重。
- 有明确目标和可执行请求。
- 没有安全风险。
- 事实基本清楚。
- 选择的渠道适合该话题。

Pause if:

- 想立刻反击、羞辱、截图群发或辞职式爆发。
- 对方正在辱骂、威胁或失控。
- 需要查证事实、合同、政策或证据。
- 用户可能因一句话带来严重职业/法律后果。

Pause script:

“这件事我很重视，但我现在需要先确认事实，避免说得不准确。我会在【时间】前回复你。”

## 3. Choose the communication channel

- **面对面/视频**: 适合复杂冲突、关系修复、敏感反馈；需要情绪稳定。
- **电话**: 适合快速降温和澄清，但不利于留证。
- **文字/邮件**: 适合正式请求、边界、确认结论、留存记录；避免长篇情绪宣泄。
- **会议**: 适合多方协同和决策；不适合公开羞辱或处理私人矛盾。

## 4. What is the real goal?

Help user choose one primary goal:

- 降温：先让对话继续得下去。
- 澄清：减少误解。
- 拒绝：保护时间、资源、价值观。
- 说服：让对方看到共同利益和低风险试点。
- 反馈：促进行为改变。
- 道歉：修复信任。
- 边界：阻止重复伤害。
- 升级：进入正式流程。
- 退出：保护安全或尊严。

If user wants “让对方后悔/难受/听话/离不开我”， redirect to ethical goals.

## 5. Controllability check

Ask:

- 我能控制什么？
- 我只能影响什么？
- 我必须接受什么？
- 如果对方不改变，我的底线行动是什么？

Example:

“你不能控制领导是否马上认可你，但可以控制：补充事实、请求具体改进建议、约定下一次验收标准、记录沟通结果。”

## 6. Boundary decision tree

- 一次性小请求：可帮则帮，说明范围。
- 反复占用：明确规则和时段。
- 侵犯尊重：立即命名行为并要求停止。
- 涉及安全或违法：退出并寻求正式支持。

Boundary formula:

“我愿意【可接受部分】，但不能【不可接受部分】。如果继续【行为】，我会【行动】。”

## 7. Feedback readiness checklist

Before giving feedback, ensure:

- 目标是改进行为，不是发泄不满。
- 有具体事实，而不是人格评价。
- 选择私下和合适时机。
- 能说清影响和期待。
- 愿意听对方解释。
- 有下一步支持或检查点。

Avoid:

- “你就是……”
- “你总是/从来……”
- 当众羞辱。
- 把一次行为上升为人格缺陷。

## 8. Persuasion readiness checklist

Before persuasion:

- 对方最在意的指标是什么？
- 对方担心的损失是什么？
- 是否有共同标准？
- 能否提出小规模试点？
- 是否承认风险和限制？
- 是否给对方保留选择空间？

Ethical persuasion is not “让对方不知不觉同意”， but “让对方在信息充分、被尊重的情况下看到价值”。

## 9. Refusal checklist

Good refusal should be:

- 清楚：不让对方误以为还有希望。
- 简短：不过度辩解。
- 尊重：承认对方需求。
- 有边界：说明不能做什么。
- 可选：如合适，提供替代方案。

Template:

“我理解你需要支持。但这周我不能接这个任务。我能做的是周五前帮你看一次提纲；如果需要完整执行，建议找项目负责人重新排优先级。”

## 10. Missing assumptions to ask if material is insufficient

When the user gives too little information, ask up to 5 focused questions:

1. 你和对方是什么关系？谁对结果有决定权？
2. 具体发生了什么？请给一句原话或一个行为事实。
3. 你希望这次沟通达成什么最低结果？
4. 你的情绪强度0-10是多少？是否能暂缓回复？
5. 有无安全、法律、HR、客户升级或重大损失风险？
