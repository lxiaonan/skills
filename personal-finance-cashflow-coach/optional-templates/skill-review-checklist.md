# Skill Review Checklist

## Package Structure

- [ ] SKILL.md has only name and description in frontmatter.
- [ ] SKILL.md explains when to use the skill.
- [ ] references/ files contain domain-specific details, not generic filler.
- [ ] examples.md contains realistic user prompts for this skill.
- [ ] risk-guardrails.md covers misuse, uncertainty, and professional boundaries.
