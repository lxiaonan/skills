# Examples

## Example 1: “我该先还债还是投资？”

### User prompt
我有信用卡欠款3万元，年化18%，每月税后收入1.2万，房租和生活费大概8000。现在有2万元现金，想投基金，还是先还债？

### Response pattern
1. **Clarify context**
   - 已知：高息信用卡18%，月结余约4000，现金2万。
   - 需补充：是否还有其他债务、应急金是否另有、工作稳定性、基金投资期限。
2. **Map facts**
   - 高息债务：3万，18%。
   - 月度现金流：收入1.2万，支出8000，理论结余4000。
   - 现金储备：2万，约2.5个月支出。
3. **Apply framework**
   - 还信用卡相当于获得18%的确定性利息节省；基金收益不确定且可能亏损。
   - 若把全部现金还债，应急金会归零；需避免再次刷卡。
4. **Options / recommendation**
   - 保守修复：保留8000–12000启动应急金，其余还信用卡；每月结余继续还。
   - 稳健优化：保留1个月必要支出，剩余还债，同时每月小额定投仅作学习。
   - 进取学习：不建议借钱或用高息债未清时追求高收益投资。
   - 推荐：优先还高息债，同时保留基本应急金。
5. **Risks**
   - 投资不能保证跑赢18%债务成本。
   - 若没有预算约束，还款后可能再次透支。
6. **Next action**
   - 本周列出所有债务APR；设置自动还款；冻结新增信用卡消费30天。

## Example 2: “买车算资产吗？”

### User prompt
我想贷款买一辆25万的车，工作通勤用，也方便见客户。算资产吗？

### Response pattern
- 先问：税后收入、首付、贷款利率和月供、保险/停车/油费、是否有业务报销或因车带来新增收入、替代方案。
- 框架：
  - 自用通勤车通常是生活工具，不是现金流资产。
  - 如果它能带来可验证新增客户收入，并且净新增收入超过总拥有成本，才可能成为业务型资产。
- 三档：
  - 保守修复：延迟购买或买低价可靠二手车，保留应急金。
  - 稳健优化：月供+总车成本控制在可承受范围，先试算3个月。
  - 进取学习：若用于业务，建立客户收入跟踪表，验证车带来的真实收入。
- 边界：不替代贷款顾问或税务师；如涉及公司报销/抵税，咨询专业人士。

## Example 3: “如何提高资产性收入？”

### User prompt
我月薪2万，但资产收入几乎没有。怎么提高资产性收入？

### Response pattern
1. 先建立现金流表，找出可用于资产栏的月度盈余。
2. 先确认应急金和高息债务情况。
3. 把“提高资产收入”拆成三类：
   - 金融资产：指数化投资、债券/货币类、股息等，需风险匹配，不荐具体产品。
   - 经营资产：小副业、自动化服务、内容/IP、可复制销售系统。
   - 能力资产：销售、会计、谈判、行业技能，提高未来现金流。
4. 30/60/90：
   - 30天：现金流盘点，确定每月资产购买预算。
   - 60天：学习一种基础投资框架或副业获客技能。
   - 90天：做一个小额、可复盘的资产或副业实验。

## Example 4: “这个副业值得投入吗？”

### User prompt
朋友让我加盟一个项目，投入5万元，说半年回本，以后躺赚。我要不要做？

### Response pattern
- 立即标记风险：承诺回本、躺赚、加盟费、朋友推荐，都需要尽调。
- 询问：合同、收入来源、是否靠拉人头、历史财务、退出条款、用户本人可承受损失、是否借钱投入。
- 框架：
  - 客户是谁？谁真正付款？
  - 单位经济是否成立？
  - 最坏情况损失多少？
  - 是否违法或接近传销？
- 三档：
  - 保守修复：不投入，先要求书面资料和第三方验证。
  - 稳健优化：只做低成本试运营，不签长期/高额合同。
  - 进取学习：若仍想做，先咨询律师/会计，设置亏损上限和退出条件。
- 禁止：不能接受“保证收益”作为依据。

## Example 5: “我收入增加了但存不下钱”

### User prompt
这两年工资从1万涨到1.8万，但还是月光，为什么？

### Response pattern
- 解释：这通常是生活方式膨胀，不是收入太低单一问题。
- 做现金流分类：涨薪流向了房租、车、餐饮、购物、订阅、社交、债务还是家庭？
- 找“耗现金项目”：新增固定支出最危险，因为它把未来收入提前锁死。
- 建议：
  - 把涨薪的一部分自动转入资产/应急金账户。
  - 设立“先支付自己”规则，但不能因此逾期债务或牺牲基本责任。
  - 选一个高泄漏类别做30天实验。
- 复盘问题：这笔支出是在买自由，还是在买更多月供？

## Example 6: “从工资依赖到资产收入计划”

### User prompt
我想5年内不那么依赖工资，怎么开始？

### Response pattern
- 先定义目标：不是立刻辞职，而是提高资产收入覆盖率。
- 指标：应急 runway、月结余率、资产收入覆盖率、技能投资进度。
- 路径：
  - 保守修复：现金流稳定和债务清理。
  - 稳健优化：自动资产购买+技能提升。
  - 进取学习：副业/经营资产小实验。
- 30/60/90：现金流表、学习计划、一个小实验、季度复盘。
