# Decision Checklists

## Universal Intake Questions
Ask only what is necessary, but before strong recommendations collect:
1. 你所在国家/地区？是否涉及税务、贷款、房产或法律合同？
2. 税后月收入和收入稳定性？
3. 每月必要支出、可选支出、储蓄/投资金额？
4. 债务清单：余额、APR/利率、最低还款、期限、是否有罚金？
5. 应急金：现金或高流动性资金能覆盖几个月必要支出？
6. 资产清单：现金、投资、养老金、房产、业务、知识产权、设备等；每月净现金流？
7. 目标：金额、时间、优先级（安全、自由、买房、教育、创业等）？
8. 风险承受度：能接受多大亏损、收入中断、波动和债务压力？
9. 家庭责任：赡养、育儿、医疗、保险、共同债务？
10. 当前决策是否需要借钱、抵押、担保、辞职、合伙或签合同？

## Monthly Cash-Flow Review Checklist
- [ ] 本月税后收入是多少？是否一次性/不稳定？
- [ ] 必要支出占收入比例是否过高？
- [ ] 可选支出中哪三项最不符合目标？
- [ ] 有无新增债务？新增原因是什么？
- [ ] 本月是否购买了产现金资产？
- [ ] 本月是否提升了未来赚钱能力？证据是什么？
- [ ] 资产收入覆盖支出比例是多少？
- [ ] 下月只改一个动作，改什么？

## Debt vs Investment Checklist
Use before answering “先还债还是投资？”
- [ ] 所有债务利率和余额明确吗？
- [ ] 是否有信用卡、网贷、 payday/现金贷、循环利息？
- [ ] 是否已有至少1个月必要支出的启动应急金？
- [ ] 投资是否需要借钱？如是，标记为高风险。
- [ ] 投资期限是否短于3年？短期限不宜承担高波动。
- [ ] 是否有雇主匹配或税优账户？需说明不是税务建议。
- [ ] 用户能承受本金亏损吗？
- [ ] 默认倾向：高息债优先于不确定投资收益。

## Car Purchase Checklist
Use for “买车算资产吗？”
- [ ] 用途：通勤必需、业务创收、家庭刚需、面子消费？
- [ ] 总拥有成本：首付、月供、利息、保险、油电、停车、维修、折旧、税费。
- [ ] 是否能增加收入或显著降低必要成本？
- [ ] 车贷月供占税后收入比例？
- [ ] 购买后应急金还剩几个月？
- [ ] 是否有更低成本替代：二手车、公共交通、租车、共享、延迟购买？
- [ ] 分类：业务净产现金可能是资产；大多数自用车是生活工具/现金流负债。

## House Purchase Checklist
Use for “房子是不是资产？” or mortgage decisions.
- [ ] 自住、出租、混合用途还是投资？
- [ ] 总成本：首付、月供、利息、物业、税费、保险、维修、装修、空置、交易税费。
- [ ] 若出租：净租金是否覆盖所有成本和空置维修储备？
- [ ] 若自住：相比租房，现金流压力如何？
- [ ] 房贷后债务压力和应急金如何？
- [ ] 是否把房价上涨当作唯一盈利来源？标记为投机。
- [ ] 是否理解当地税法、贷款合同、产权和流动性风险？需要专业人士。

## Side-Hustle Checklist
- [ ] 目标是学习、现金流、转主业还是兴趣？
- [ ] 需要投入多少钱和多少小时？最大可承受损失？
- [ ] 是否需要借债、囤货、加盟费、拉人头？如有，谨慎并检查合法性。
- [ ] 是否有付费客户证据，而不仅是“感觉有市场”？
- [ ] 单位经济模型：价格、成本、毛利、获客成本、回本时间。
- [ ] 会不会影响主业、健康、家庭、信用？
- [ ] 30天内最小验证实验是什么？
- [ ] 何时停止、何时扩大？预设指标。

## Investment Readiness Criteria
Before discussing investment options, check:
- [ ] 用户了解投资可能亏损，且没有要求保证收益。
- [ ] 有基本应急金或稳定现金流。
- [ ] 高息债务已被控制或有明确还款计划。
- [ ] 投资期限、流动性需求、风险承受度明确。
- [ ] 不使用生活费、短期刚需资金或高息借款投资。
- [ ] 能解释投资标的如何赚钱、主要风险是什么、退出条件是什么。
- [ ] 涉及复杂产品、杠杆、房产、税务或证券建议时，建议咨询持牌专业人士。

## Missing-Data Provisional Answers
When user gives little data, use these conservative defaults:
- 高息债务通常优先于追求不确定收益。
- 自用车通常不是现金流资产，而是生活工具和负债性支出。
- 自住房不自动等于资产；先看每月净现金流、机会成本和负债压力。
- 副业先用低成本验证需求，不先重资产投入。
- 投资前先补现金流表、应急金、风险承受度和基础财务知识。
